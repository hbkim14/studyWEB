HTML 이란?

HTML은 HyperText Markup Language로 웹페이지에 해당 내용을 출력하기 위한 전자 문서 양식이다. 우리가 흔히 아는 문서 양식(.hwp, .docx, .txt)을 가진 파일들을 해당 응용 프로그램(한글, word, 메모장)으로 확인할 수 있는 것처럼, HTML 파일(.html)은 WEB 서비스를 활용하여 그 내용을 확인할 수 있다. HTML은 주로 웹페이지의 화면을 구성하는데에 활용되는데 이를 위하여 전세계적으로 미리 약속된 몇 가지 태그(tag)를 활용한다.

HTML 태그

지금까지 배운 HTML 태그 요약!

<!DOCTYPE html> : 현재 문서가 html로 쓰여진 문서임을 나타냄

<html></html> : head 태그와 body 태그를 묶어줌

<head></head> : 해당 태그 내부에 웹페이지 특성과 관련된 정보(태그)가 모여있음

<title></title> : 웹페이지를 띄웠을 때 tab바에 출력되는 웹페이지 이름 설정

<meta charset="" /> : 웹페이지의 문자 저장 규칙(charset)을 알려줌

<body></body> : 해당 태그 내부에 웹페이지에 출력할 정보(태그)가 모여있음

<h1></h1> ~ <h6></h6> : 제목 태그, h 뒤의 숫자가 커질수록 글자크기가 작아짐

<p></p> : 단락 태그, 해당 태그 내부의 정보들이 하나의 단락이 됨

<u></u> : 밑줄 태그, 해당 태그로 감싸진 텍스트에 밑줄 출력

<strong></strong> : 강조 태그, 해당 태그로 감싸진 텍스트를 굵은 글씨로 출력

<br> : 줄바꿈 태그

<ul>       : 리스트 부모 태그, 해당 태그에 감싸진 목록을 같은 양식으로 출력

  <li></li> : 리스트 자식 태그, 해당 태그에 감싸진 텍스트를 목록 요소로 출력

</ul>

<ol></ol> : 리스트 부모 태그, 해당 태그에 감싸진 목록을 순서를 부여하여 출력 (ordered list)

<a href="(클릭 )" title="(힌트제공)" target="_blank"></a> : 링크 태그(anchor), 클릭 시 미리 저장된 주소로 이동
href : 클릭 시 이동할 웹페이지 주소 저장, title : 마우스를 올릴 시 힌트 제공, target="_blank" : 새로운 창을 띄워 해당 주소로 이동
